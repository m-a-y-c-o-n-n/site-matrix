Função Afim:

A função afim, também chamada de função do 1º grau, é uma função f : ℝ→ℝ, definida como f(x)= ax + b,sendo a e b números reais.

Exemplo:

Construa o gráfico da função f (x) = 2x + 3.

Para construir o gráfico desta função, vamos atribuir valores arbitrários para x, substituir na equação e calcular o valor correspondente para a f (x). Sendo assim, iremos calcular a função para os valores de x iguais a: - 2, - 1, 0, 1 e 2.

Substituindo esses valores na função, temos:

f (- 2) = 2. (- 2) + 3 = - 4 + 3 = - 1

f (- 1) = 2 . (- 1) + 3 = - 2 + 3 = 1

f (0) = 2 . 0 + 3 = 3

f (1) = 2 . 1 + 3 = 5

f (2) = 2 . 2 + 3 = 7

Identificação de função atravez do gráfico:

Para determinar a lei de formação de uma função através do gráfico, precisamos conhecer o tipo da função (se for uma das básicas) ou ter alguma informação extra sobre sua expressão se for uma função desconhecida.

Uma vez que tivermos uma fórmula, devemos impor as condições do gráfico, substituindo o x e o y=f(x) para cada ponto que pertence a função. Isso nos dará um sistema, possivelmente linear, que permitirá determinar os parâmetros e encontrar a expressão da função

Exemplo:

Como o gráfico é uma reta, a função f tem que seguir a expressão f(x) = ax + b

Pelo gráfico, vemos que a reta passa no ponto A = (3,5) . Isto quer dizer que quando x = 3 então o f(x) = 5 . Ou seja, vamos impor esta condição substituindo o x por 3 e o f(x) por 5 . Aí escrevemos:

f(x) = ax + b

5 = a⋅3 + b

5 = 3a + b

E aí temos nossa primeira equação.

Fazendo a mesma coisa para o ponto B=(−4,0) , obtemos:

f(x) = ax + b

0 = a⋅(−4) + b

0 = -4a + b

E aí temos nossa segunda equação. Ou seja, chegamos no sistema abaixo:

3a + b = 5 (I)

-4a + b = 0 (II)

resolvendo pelo metodo de subtração (adição).Fazendo pela diferença da primeira pela segunda fica :

(3a - (-4a)) + (b - b) = 5 - 0

3a + 4a + 0 = 5

7a = 5

a = 5/7

com o a = 5/7 , podemos encontrar o valor de b substituindo na segunda equação:

-4a + b = 0

-4 . 5/7 + b = 0

b = 5/7 x + 20/7

Então a expressão da função f é:

f(x) = 5/7 x + 20/7
